Длительность звука

У любой ноты есть продолжительность. Указать продолжительность можно с помощью множества разных способов. С помощью секунд, музыкальной длительности или же с помощью специального языка длительности, который используется в Тоне.

Немузыкальная длительность

Числа

Числовые значения переводятся в секунды

1.2 = 1.2 секунда

3 = 3 секунды

Частоты

Можно передовать значения в герцах

1hz = 1 секунда

5hz = 0.2 секунды

Время в музыке

В музыке есть и свое разделение времени. Оно делится на четверти, восьмые, целые и шестнадцатые. Каждое деление изображает разную длительность и имеет свое названия, для того, чтобы музыканты могли легче понимать друг-друга.

Здесь можно подробнее прочитать про то, как устроено время в музыке

Тайминги в Tone.js

Что такое BPM?

У каждого музыкального произведения есть единый параметр скорости — Beats Per Minute. Например вальс играется в скорости 70 BPM, а хаус использует 100 BPM, хардкор-техно играет со скоростью от 200 BPM.

Время относительно BPM (Transport time)

Чтобы создавать секвенции и мелодии в мелодии есть Transport Time. Также можно оперировать делением на такт, четверть и шестнадцатые.

"32:0:0" = Начало 32 такта

"4:3:2" = 4 такт + 3 четверть + 2 шестнадцатых.

"1:2" = 1 такт + 2 четверть (шестнадцатая может быть пропущена)

Нотная нотация

Тут все как в музыкальной теории.

C = До
C# = До Диез
D = Ре
D# = Ре Диез
E = Ми
F = Фа
F# = Фа Диез
G = Соль
G# = Соль Диез
A = Ля
A# = Ля Диез
B = Си
